# 适配前端组件的结构化文案（TS类型+文案分离）
## 1. 类型定义（TypeScript）
```ts
interface TagItem {
  label: string;
}

interface OverviewCard {
  mainNumber: string;
  numberDesc: string;
  content: string;
  tags: TagItem[];
}

interface FeatureCard {
  title: string;
  content: string;
  tags: TagItem[];
}

interface ProjectAchievementData {
  topOverview: OverviewCard;
  bottomCards: FeatureCard[];
}
```

# 布局结构说明（前端开发参考）
1.  **整体容器**：全屏宽度，深色背景，上下分两个区块，区块间留间距
2.  **顶部概览区**：
    - 通栏卡片，内部分为「数字区」和「内容区」左右布局（或上下布局，适配移动端）
    - 数字区：大字号主数字 + 小字号说明文字
    - 内容区：正文描述 + 横向标签栏
3.  **底部卡片区**：
    - 3列等宽网格布局（响应式：大屏3列、平板2列、移动端1列）
    - 单卡片结构：标题（大字号粗体）→ 正文（常规字号）→ 底部横向标签栏
    - 卡片间留统一间距，卡片内边距统一
